  <name>Task 1: Build ChatPanel component with history loading, message sending, and action display</name>
  <files>frontend/components/ChatPanel.tsx, frontend/__tests__/ChatPanel.test.tsx</files>
  <behavior>
    - Calls fetchChatHistory on mount and renders loaded messages
    - Parses JSON string actions field from history into ChatActions objects
    - Renders each message using ChatMessage component with ChatActionCard slotted as children
    - Shows a loading indicator (pulsing dots) while waiting for AI response
    - On send: adds optimistic user message to list immediately, calls sendChatMessage, adds assistant response
    - After response with trades: calls refreshPortfolio()
    - After response with watchlist_changes: calls refreshWatchlist()
    - Auto-scrolls to bottom when new messages are added
    - Disables input while loading (prevents double-send)
    - Shows error message in chat if sendChatMessage throws
    - Renders a header bar with "AI Assistant" title and a close button (calls onClose prop)
  </behavior>
  <action>
    Build ChatPanel as a stateful component that orchestrates the chat experience.

    Props: `onClose: () => void` (called when close/X button is clicked).

    **State:**
    - `messages: ChatMessage[]` -- local state, not global context
    - `loading: boolean` -- true while awaiting API response
    - `scrollRef: useRef<HTMLDivElement>` -- for auto-scroll

    **On mount (useEffect):**
    - Call `fetchChatHistory()`
    - Map each history message: if `actions` is a string, `JSON.parse(actions)` into ChatActions; if null, leave null
    - Set messages state with the parsed array

    **handleSend callback:**
    1. Guard: if input empty or loading, return
    2. Create optimistic user message: `{id: crypto.randomUUID(), role: 'user', content: text, actions: null, created_at: new Date().toISOString()}`
    3. Append to messages, set loading true
    4. Try: call `sendChatMessage(text)`
    5. On success: create assistant message from response, with `actions` built from response.trades and response.watchlist_changes (if either non-empty array, create ChatActions object; else null)
    6. Append assistant message to messages
    7. If `response.trades.length > 0`: call `refreshPortfolio()`
    8. If `response.watchlist_changes.length > 0`: call `refreshWatchlist()`
    9. On error: append an error assistant message ("Sorry, something went wrong. Please try again.")
    10. Finally: set loading false

    **Auto-scroll (useEffect on [messages]):**
    - `scrollRef.current.scrollTop = scrollRef.current.scrollHeight`

    **Rendering layout (full height flex column):**
    ```
    <div className="flex flex-col h-full bg-terminal-surface border-l border-terminal-border">
      {/* Panel header */}
      <div className="flex items-center justify-between px-3 py-2 border-b border-terminal-border shrink-0">
        <span className="text-terminal-yellow text-sm font-mono font-semibold">AI Assistant</span>
        <button onClick={onClose} className="text-terminal-muted hover:text-terminal-text text-lg" aria-label="Close chat">x</button>
      </div>

      {/* Message list - scrollable */}
      <div ref={scrollRef} className="flex-1 overflow-y-auto p-3 space-y-3" data-testid="chat-messages">
        {messages.map(msg => (
          <ChatMessage key={msg.id} message={msg}>
            <ChatActionCard actions={msg.actions} />
          </ChatMessage>
        ))}
        {loading && <LoadingIndicator />}
      </div>

      {/* Input area */}
      <ChatInput onSend={handleSend} disabled={loading} />
    </div>
    ```

    **LoadingIndicator:** A simple inline component rendering three pulsing dots:
    ```
    <div className="flex items-center gap-1 px-3 py-2" data-testid="chat-loading">
      <div className="w-2 h-2 bg-terminal-blue rounded-full animate-bounce" style={{animationDelay: '0ms'}} />
      <div className="w-2 h-2 bg-terminal-blue rounded-full animate-bounce" style={{animationDelay: '150ms'}} />
      <div className="w-2 h-2 bg-terminal-blue rounded-full animate-bounce" style={{animationDelay: '300ms'}} />
    </div>
    ```

    **Tests:** Mock `sendChatMessage`, `fetchChatHistory` from api.ts and `usePriceContext` from PriceContext (following the TradeBar.test.tsx mock pattern). Test:
    1. Renders and loads history on mount (mock fetchChatHistory returning 2 messages)
    2. Parses JSON string actions from history correctly
    3. Shows loading indicator while API call is in flight
    4. Sends message and displays assistant response
    5. Calls refreshPortfolio when response has trades
    6. Calls refreshWatchlist when response has watchlist_changes
    7. Disables input while loading
    8. Shows error message on API failure
    9. Renders close button that calls onClose
  </action>

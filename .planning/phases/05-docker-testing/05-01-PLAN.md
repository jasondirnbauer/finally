<!-- Key path resolution in main.py that affects Docker static file serving -->
From backend/app/main.py:
```python
_static_dir = Path(__file__).parent.parent / "static"
if _static_dir.is_dir():
    app.mount("/", StaticFiles(directory=str(_static_dir), html=True), name="static")
```
Note: __file__ is /app/backend/app/main.py, so .parent.parent = /app/backend/, making _static_dir = /app/backend/static/
The Dockerfile COPY puts frontend output at ./static/ from WORKDIR /app/backend, so path = /app/backend/static/. This SHOULD be correct.

From Dockerfile:
```dockerfile
WORKDIR /app/backend
COPY --from=frontend-build /app/frontend/out ./static/
# This produces /app/backend/static/ -- matches main.py's resolution
```
